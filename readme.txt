== Description ==

Shortcodes to render crypto currency exchange rates and an crypto currency exchange rates calculator.

Important:
Please note that this plugin relies on an external api operated by LAMP solutions GmbH.
To view the terms of service and privacy policy for this api, please visit:
[https://cryptocheck.api.cryptopanel.de/api/documentation](https://cryptocheck.api.cryptopanel.de/api/documentation)

Example shortcodes:

`[cpcc-calculator]`
`[cpcc-currency from="EUR" to="BTC" amount="100,00"]`
`[cpcc-currency from="EUR" to="Dash" amount="100,00"]`
